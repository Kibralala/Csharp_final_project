# WoW Guild Management

This is my final project for my Object-Oriented Programming course. It is a **web application for managing a World of Warcraft guild**, designed to practice C# and web development skills while having some fun with a fantasy theme.

## Features

- Guild management (members, characters, roles)
- Raid management
- Raid signups
- User authentication
- Responsive UI
- CRUD operations for guild data

## Technologies Used

- **C# / ASP.NET Core MVC**
- **Entity Framework Core**
- **SQL Server**
